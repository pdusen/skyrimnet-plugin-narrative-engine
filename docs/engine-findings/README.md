# Engine findings

Notes on the actual runtime behavior of SKSE, CommonLibSSE-NG, and the
Skyrim engine surface we build against — the kind of thing that isn't
documented anywhere authoritative, that we learned the hard way, and that
we don't want to learn again.

Each file is one focused finding: what we observed, what the right
incantation is, and what the false leads were so they don't get re-tried.
Cross-reference from code comments when relevant.
